# House Price Prediction

A machine learning web application that predicts house prices based on:

* House Size (sqft)
* Number of Bedrooms

## Tech Stack

### Backend

* Python
* FastAPI
* Scikit-Learn
* NumPy

### Frontend

* React.js
* Axios

## Features

* Single House Prediction
* Batch Prediction
* REST API
* Responsive User Interface

## Run Backend

```bash
uvicorn main:app --reload
```

## Run Frontend

```bash
npm install
npm start
```

## API Endpoints

POST /predict

POST /predict_batch
